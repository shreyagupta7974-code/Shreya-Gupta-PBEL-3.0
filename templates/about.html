About Resume Screening Tool

Resume Screening Tool is a web application developed to help students and job seekers analyze their resumes quickly. It automatically extracts important information from a PDF resume, evaluates the resume quality, calculates an ATS score, and provides useful suggestions to improve the resume.

Project Features

  • ✔ Upload Resume (PDF)
  • ✔ Resume Text Extraction
  • ✔ Candidate Information Detection
  • ✔ Technical Skills Detection
  • ✔ Education Detection
  • ✔ Project Detection
  • ✔ Experience Detection
  • ✔ Certification Detection
  • ✔ ATS Score Calculation
  • ✔ Resume Quality Analysis
  • ✔ AI-Based Suggestions

Technologies Used

  • Python
  • Flask
  • HTML5
  • CSS3
  • PyMuPDF (fitz)
  • PyPDF
  • Regular Expressions (Regex)

Project Objective

The main objective of this project is to simplify the resume screening process by automatically analyzing resumes and providing meaningful feedback. This helps users improve their resumes before applying for internships or jobs.
